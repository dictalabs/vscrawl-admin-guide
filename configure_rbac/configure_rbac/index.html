Why Start with RBAC

After the first login, it is recommended to:

  1. Create a new role.
  2. Assign the role to a new operator.
  3. Perform any remaining administrative operations using the newly created non-default administrator account.

This approach ensures enhanced security by minimizing reliance on the default administrator account.
